# Carpeta de IA: Rules, Agents y Prompts

## Objetivo
Esta carpeta organiza el uso de IA de forma responsable y reutilizable dentro del proyecto.

## Estructura
- `rules/`: reglas que la IA debe respetar.
- `agents/`: perfiles especializados para distintas tareas.
- `prompts/`: instrucciones reutilizables para generar documentación, código, pruebas o diagnósticos.

## Buenas prácticas
- No usar IA como sustituto de criterio técnico.
- Validar siempre el resultado generado.
- Mantener coherencia entre IA, documentación y código.
- Registrar decisiones importantes derivadas del uso de IA.
